# Calculator-Project
## Overview
_This is a simple web-based calculator that can perform basic arithmetic operations, as well as more advanced operations like exponentiation, square root, and squaring a number. It supports chaining calculations_

## Features
- Addition (+), Subtraction (-), Multiplication (*), Division (/)
- Exponentiation (^)
- Square root (√)
- Squared (x²)
- Clear (C) button to reset all input
- Delete (DEL) button to remove the last entered number/operator
- Supports decimal numbers
- Chaining: continue calculations after pressing =

## How to Use
_Click on the number buttons (0–9) or decimal point (.) to type numbers. Click on one of the operator buttons (+, -, *, /, ^, √, x²).For √ or x², only the first number is used. After selecting an operator, type the second number. Click the = button to see the result. Click C to reset all input.Click DEL to delete the last digit or operator. After getting a result, you can immediately select an operator to continue calculations._

## Notes
- Division by zero will throw an error.
- The display shows the current input and operator before you press =.
- The result is stored in memory, allowing you to chain further calculations without manually retyping it.

## File Structure
- index.html – Contains the calculator layout and buttons.
- style.css – Styles for the calculator interface.
- index.js – Contains all JavaScript logic for handling input, calculations, and display updates.
